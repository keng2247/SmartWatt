# Code of Conduct

## Purpose

SmartWatt contributors are expected to keep the project welcoming, professional, and respectful.

## Expected Behavior

- Communicate respectfully and constructively.
- Focus on technical outcomes and collaborative problem solving.
- Accept feedback professionally.
- Respect differing viewpoints and experience levels.

## Unacceptable Behavior

- Harassment, insults, or discrimination.
- Personal attacks or intimidation.
- Publishing private information without permission.
- Any behavior that undermines a safe collaboration environment.

## Enforcement

Project maintainers are responsible for interpreting and enforcing this policy. They may remove, edit, or reject comments, commits, issues, or pull requests that violate these standards.

## Scope

This policy applies in repository discussions, issues, pull requests, and project communication channels.
